# seller.py
# Программа для обновления информации об остатках товара и цен на Ozon. #

## Как установить ##
Для начала Вам нужно зарегистрироваться как продавец на [Оzon](https://ozon.ru/) и [Timeworld](https://timeworld.com/). Для запуска программы необходим ключ доступа (токен) продавца и клиент айди.

Python3 должен быть уже установлен. 

## Как использовать ##
### Получение данных ###
Программа скачивает список имеющихся товаров в количестве 1000 шт за один раз. из маркетплейса Озон и список товаров с данными об остатках ассортимента часов Casio c сайта Timeworld.
### Обновление остатков ###
Затем сравнивает эти списки между собой опираясь на артикулы(id) товаров и создаёт новый список в котором будут указанно количество оставшегося товара. Если артикулы товаров совпадают, то обновляет остаточное количество используя список с Timeworld по схеме:
1. Если остатков больше 10, то в новом списке количество устанавливает 100.
2. Если остался 1 товар, то в новом списке 0.
3. В любом другом случае остается фактическое количество из остатка.
По итогу если остаётся товар которого нет в списке, он добавляется с остаточным количеством - 0. Новый список отправляется на маркетплейс Озон и меняет информацию об остаточном товаре.
### Обновление цен ###
Далее программа обновляет цены.

Берет список товаров с Timeworld сравнивает его с товарами которые уже есть на Озоне, и переносит цену округляя её (отбрасывая копейки) создаёт новый список. Новый список частями отправляется на Озон и обновляет цены.



# market.py
# Программа для обновления информации об остатках товара и цен на Yandex market. #

Работает по двум моделям логистики FBS* и DBS* см. примечание.

## Как использовать ##
### Получение данных ###
Программа скачивает список имеющихся товаров в количестве 1000 шт за один раз. из маркетплейса Yandex market и список товаров с данными об остатках ассортимента часов Casio c сайта Timeworld.

### Примечание ###

FBS (Fulfillment by Seller) — "Выполнение продавцом".
Продавец:
1. Хранит товар у себя.
2. Сам упаковывает.
3. Сам отправляет в пункт приёма Яндекс Маркета.

DBS (Delivery by Seller) — "Доставка продавцом".
Продавец:
1. Хранит товар у себя.
2. Сам упаковывает.
3. Сам доставляет.

SKU (Stock Keeping Unit) — это уникальный идентификатор конкретного товара.
На Яндекс Маркете SKU = карточка товара или конкретная вариация товара.
